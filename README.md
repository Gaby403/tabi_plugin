# Tabi Plugins by Franklyn

A WordPress plugin developed to add specific design and interactivity functionalities, including custom button styles, Elementor Swiper adjustments, and a configurable Popup system.

## 🚀 Features

### 1. Customizable Popup
A lightweight popup system fully manageable via the admin panel.
*   **Global or Specific Activation**: Choose to display on the entire site or only on specific pages (by ID).
*   **Editable Content**: Customizable text and action button (with link).
*   **Design**: Integrated background color picker.
*   **Positioning**: Option to display in the center or at the bottom of the screen.
*   **Behavior**: Define a delay or scroll trigger for display after page load.
*   **Cookie Control**: Once closed, the popup does not reappear for 24 hours for the same user.

### 2. Button Styling
Loads custom CSS styles for CTA (Call to Action) buttons specifically for mobile devices.
*   **Target IDs**: `#btn-cta-mobile`, `#btn-cta-mobile2`.

### 3. Elementor Swiper Adjustments
Applies fixes and custom styles to the navigation (arrows) of the Elementor Swiper/Carousel widget.
*   Improves positioning and size of arrows on mobile and desktop.

## 📦 Installation

1.  Download the plugin `.zip` file.
2.  In the WordPress dashboard, navigate to **Plugins > Add New**.
3.  Click **Upload Plugin** and select the downloaded file.
4.  Click **Install Now** and then **Activate**.

## ⚙️ Popup Configuration

Access the **Popup Settings** menu in the WordPress sidebar.

### Display and Control
*   **Enable Popup**: Check to enable the popup on the site.
*   **Page IDs**: Enter page IDs separated by commas (e.g., `10, 25`) to display only on them. Leave blank to display on all.
*   **Popup Position**: Choose between "Center of Screen" or "Bottom of Screen".
*   **Trigger Type**: Choose between "Scroll Percentage" or "Time Delay".

### Content and Style
*   **Popup Text**: The main message of the popup.
*   **Action Button**: Button text (e.g., "Learn More") and destination URL.
*   **Background Color**: Background color of the popup window.
